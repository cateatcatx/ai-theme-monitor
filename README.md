# A股 AI 主题 · 波段监控

监控A股AI主题的**波段状态**(5-20日窗口, 目标捕捉10个交易日左右的波段), 每个交易日自动更新, 打开 `index.html` 即可查看。

## 日常使用

- **看指标**: 双击 `index.html` (纯本地页面, 无需服务器)
- **手动更新**: 双击 `run_update.bat`, 或 `python -X utf8 update_data.py`
- **自动更新**: Windows计划任务 `AI_Theme_Monitor_Daily` 每个交易日 17:30 运行(仅当天开机在线时执行); 运行日志见 `update.log`

## 在线版 (GitHub Pages, 免费自动更新)

- 网页托管在 GitHub, **不依赖本机开机**, 手机加书签即可随时看; 网址形如 `https://<用户名>.github.io/<仓库名>/`。
- `.github/workflows/deploy.yml` 每交易日定时 (北京 18:30 / 22:00) 在 GitHub 服务器上跑 `update_data.py` 重算 `data.js`, 连同页面一起发布; 也可在仓库 Actions 页面手动 "Run workflow"。
- 若中证官网从 GitHub 美国服务器不可达, 脚本会自动回退到仓库里缓存的 `data/*.csv`(因此这些缓存需随仓库提交), 页面仍能发布, 只是数据停留在上次成功抓取。

## 指标体系

锚定指数: **中证人工智能主题 930713**, 基准沪深300, 分母中证全指。

**1. 波段方向分 (0-100)** —— 波段方向与强度
- 趋势 40%: 收盘对10日线乖离(±6%映射) + MA5>MA10 + MA10向上(5日)
- 相对动量 35%: 相对沪深300的10日超额收益(±8%映射) + RS线站上其10日均线
- 广度 25%: AI产业链7个指数(人工智能/半导体/通信设备/计算机/云计算/机器人)中收于10日线上方的比例
- ≥60 波段上行 · <40 波段破位

**2. 波段温度 (0-100)** —— 过热/冰点
- 拥挤 60%: 主题成交额 ÷ 中证全指成交额, 5日平滑, 滚动1年百分位
- 涨速 40%: 指数10日涨幅的滚动1年百分位
- ≥80 过热(止盈区) · ≤30 冰点(酝酿区)

**状态机 (方向 × 温度)**: 上行健康(绿) / 上行过热(橙) / 转折震荡(黄) / 下行(红) / 冰点酝酿(蓝)。

**波段信号** (标记在图上, 每日随数据更新):
- 入场 = 方向分从下方上穿60, 且当日温度<70
- 离场 = 方向分跌破40(破位), 或 温度≥85且收盘跌破10日线(过热破线止盈)

回测(信号日收盘成交, 不计滑点): 2019年来63笔, 胜率41%, 单笔中位-1.3%/均值+2.5%, 持有中位9日, 最差单笔-6.4%; 2022年来34笔, 胜率50%, 均值+3.6%, 持有中位10日。典型趋势跟随画像: 小亏多次+大赚几次(2024/09 +29.5%, 2025/07 +37.5%, 2026/04 +22.1%), 纪律性执行离场是关键。T+1收盘成交的结果同量级(胜率反而略升)。

规则经多变体对比选定(入场阈值55/60/65、温度上限70/80、离场40/45/50、过热线80/85、广度过滤、连续确认等), 该组合在2019+与2022+两个时段的累计收益、胜率、持有期均衡最优。

## 数据源

中证指数官网 `csindex.com.cn/csindex-home/perf/index-perf` (免费无鉴权, 含OHLC/成交额/PE, T日数据当晚更新)。
东财/akshare接口在本机被拦截不可用。原始数据缓存在 `data/*.csv`, 拉取失败时自动回退缓存。

## 文件

| 文件 | 作用 |
|------|------|
| `update_data.py` | 拉数据 + 算指标 + 波段信号回测 + 写 `data.js` |
| `index.html` | 展示页(ECharts本地化, 双击即开) |
| `data.js` | 页面数据(自动生成) |
| `run_update.bat` | 一键更新(计划任务入口) |
| `data/` | 各指数日线缓存 |

## 局限

- 信号胜率只有四到五成, 靠盈亏比赚钱: 必须机械执行离场, 扛单会把这套系统的期望值变负
- 930713聚焦AI核心公司(算力+应用), 不完全覆盖"AI+"扩散行情; 产业链表可辅助定位扩散位置
- 中证官网偶发延迟更新, 页面会显示T-1数据; 计划任务在关机/断网时跳过该日, 补跑双击 `run_update.bat`
